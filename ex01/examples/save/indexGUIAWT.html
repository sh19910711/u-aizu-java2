Problem 2:  An Applet Handling Action Events (30 out of 100)

The details of Button, ActionEvent and ActionListener

View the answer of this problem by "appletviewer ExGUIAWT_02_ans.html".

    This is an extension of ExGUIAWT_01. Clicking one of the buttons on the window can change the message's color.

    Requirements:

     - class name: ExGUIAWT_02 (file name: ExGUIAWT_02.java)

     - super class: ExGUIAWT_01 (If you have not finished Problem 1, please use ExGUIAWT_01_ans instead of ExGUIAWT_01, as the super class)

     - implement interface: ActionListener [This is to catch button-clicking events (Action events). Since any object can be an action listener by implementing the ActionListener interface, here we use ExGUIAWT_02 as a action listener.]

    Comments:

     - Declare attributes to save the three buttons and the message color.

     - Override Applet's init() method:
           (1) Place the buttons on the Applet (use add());
           (2) Add action listener to each button (use addActionListener(this)). That is, we use one listener to catch events from 3 buttons.

     - Override paint() method:
          (1) Call super class's paint() method;
          (2) Set the message color attribute to the Graphics object (use setColor());
          (3) Draw a message "You have selected *** color!".

     - Implement the actionPerformed() method for ActionListener
         (1) Get the source object on which the event occurred (call getSource() of the argument).
         (2) The source object should be one of the three buttons saved in the attributes. Therefore, you can know which button has been clicked by comparing the button attributes with the source object.
         (3) Decide which color should be used (save the color to the message color attribute).
         (4) Finally, call repaint() method of itself.

Problem 3: An Applet Handling Mouse Events (20 out of 100)

The details of MouseListener, MouseMotionListener and MouseEvent.

View the answer of this problem by "appletviewer ExGUIAWT_03_ans.html".

    This is an extension of ExGUIAWT_02. By dragging the mouse on the window, you can draw a line segment.

    Requirements:

     - class name: ExGUIAWT_03 (file name: ExGUIAWT_03.java)

     - super class: ExGUIAWT_02 (If you have not finished Problem 2, please use ExGUIAWT_02_ans instead of ExGUIAWT_02, as the super class)

     - implement interface: MouseListener (to catch mouse button on/off events),  MouseMotionListener (to catch mouse motion events). Since any object can be an listener, here we use ExGUIAWT_03 itself to make those listeners.]

    Comments:

     - Use attributes to save mouse pointer positions (the two end points of the line segment).

     - Use an attribute value (boolean type) to save dragging status (true or false).

     - Override Applet's init() method:
           (1) Call the same method of its supper class;
           (2) Add listeners (this) to the ExGUIAWT_03 object itself (use addMouseListener() and addMouseMotionListener()). That is, we use two listeners to catch events on one object.

     - Override Applet's paint() method:
          (1) Call the paint() method of super class;
          (2) If the dragging status attribute is "true", draw a line segment based on the values of the attributes;

     - Implement methods mousePressed() and mouseReleased() for MouseListener:
         (1) Change the dragging status attribute.
         (2) In mousePressed(), save mouse position to the attribute as one end point of the line segment
         (3) Call repaint() method.

     - Implement method mouseDragged() for MouseMotionListener:
         (1) Save mouse position to the attribute as another end point of the line segment
         (2) Call repaint() method.

     - Implement the following methods but do nothing:
         mouseClicked(),  mouseEntered(), mouseExited(), and mouseMoved().
