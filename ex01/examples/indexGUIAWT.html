[Step 2]  An Applet Handling Action Events (30points)

The details of Button, ActionEvent and ActionListener

View the answer of this problem by "appletviewer ExGUIAWT_02_ans.html".

    This is an extension of ExGUIAWT_01. Clicking any button can change the number between the buttons.

    Requirements:

     - class name: ExGUIAWT_02 (file name: ExGUIAWT_02.java)

     - super class: ExGUIAWT_01 (If you have not finished Step 1, please use ExGUIAWT_01_ans instead of ExGUIAWT_01, as super class)

     - implement interface: ActionListener [This is to catch button-clicking events (Action events). Since any object can be an action listener by implementing the ActionListener interface, here we use ExGUIAWT_02 as a action listener.]

    Comments:

     - Declare attributes to save the buttons (Button objects), the number and a label (Label object) for showing the number.
    - The number and the label should be declared as protected attributes for subclasses.

     - Override Applet's init() method:
           (1) call super class's init() method.
           (2) Place the buttons and the label on the Applet (use add());
           (3) Add action listener (this) to each button (use addActionListener(this)). That is, we use one listener to catch events from all the buttons.

     - Implement the actionPerformed() method for ActionListener
         (1) Get the source object on which the event occurred (call getSource() of the argument).
         (2) The source object should be one of the buttons saved in the attributes. Therefore, you can know which button has been clicked by comparing the button attributes with the source object.
         (3) Increase or decrease the number according to the clicked butten.
         (4) Set the number to the label (call Label setText() method).

[Step 3] An Applet Handling Mouse Events (30points)

The details of MouseListener, MouseMotionListener and MouseEvent.

View the answer of this problem by "appletviewer ExGUIAWT_03_ans.html".

    This is an extension of ExGUIAWT_02. By horizontally dragging the mouse on the window, you can chagne the number quickly.

    Requirements:

     - class name: ExGUIAWT_03 (file name: ExGUIAWT_03.java)

     - super class: ExGUIAWT_02 (If you have not finished Step 2, please use ExGUIAWT_02_ans instead of ExGUIAWT_02, as the super class)

         * If you use ExGUIAWT_02 _ans as the super class, use the information below for accessing its attributes:
                   protected Label l_num = new Label("0");
                   protected int number = 0;

     - implement interface: MouseListener (to catch mouse button on/off events),  MouseMotionListener (to catch mouse motion events). Since any object can be an listener, here we use ExGUIAWT_03 itself to make those listeners.]

    Comments:

     - Use an attribute to save mouse x position.

     - Override Applet's init() method:
           (1) Call the same method of its supper class;
           (2) Add listeners (this) to the ExGUIAWT_03 object itself (use addMouseListener() and addMouseMotionListener()). That is, we use two listeners to catch events on one object.

     - Implement methods mousePressed() for MouseListener:
         (1) Get mouse x position (call getX() of the argument), save it to the attribute.
    - Implement method mouseDragged() for MouseMotionListener:
         (1) Get mouse x position; Compare it with the attribute (the saved position) to decide how to change the number in super class.
         (2) Set the number to the label in the super class.
         (3) Save current mouse position to the attribute for further use.
    - Implement the following methods but do nothing:
         mouseClicked(),  mouseEntered(), mouseExited(), mouseReleased(), and mouseMoved().
